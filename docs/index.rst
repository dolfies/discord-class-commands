Welcome to discord-class-commands
==================================

.. image:: /images/snake.svg
.. image:: /images/snake_dark.svg

discord-class-commands is an extension module for discord.py that facilitates class-based creation of Discord application commands.

**Features:**

- Modern Pythonic API using ``async``\/``await`` syntax
- Easy to use with an object oriented design
- Optimised for both speed and memory
- Fully compatible with discord.py's application command implementation without monkey-patching
